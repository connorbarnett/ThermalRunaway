dragged:

Determines whether the view has been swiped beyond two thresholds- 1) The view can be swiped sufficiently far to the left, casting a down vote 2) The view can be swiped sufficeintly far to the right, casting an up vote

- (void)dragged:(UIPanGestureRecognizer *)gestureRecognizer

Discussion

Determines whether the view has been swiped beyond two thresholds- 1) The view can be swiped sufficiently far to the left, casting a down vote 2) The view can be swiped sufficeintly far to the right, casting an up vote

If either of these events occur, a vote is recorded and a new company card (if available) replaces the old one If not, the view snaps back into place. Either way, Google Analytics monitors the interaction.

@param gestureRecognizer
